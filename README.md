# Flutter Invoice App

A professional, full-featured invoicing application built with Flutter.

## Features

- **Invoice Management**: Create, view, and manage invoices with ease.
- **PDF Generation**: Generate professional PDF invoices for clients.
- **Company Management**: Manage client/company details (B2B focused).
- **Offline First**: Uses Hive for fast, local storage.
- **State Management**: Built with Riverpod for robust and scalable state management.
- **Clean Architecture**: Organized for maintainability and testability.

## Technology Stack

- **Framework**: Flutter
- **State Management**: Riverpod
- **Local Database**: Hive
- **PDF Generation**: `pdf` and `printing` packages
- **Icons**: Material Design Icons

## Getting Started

1.  **Clone the repository**:
    ```bash
    git clone git@github-ismail:ismailfarisi/invoice_app.git
    ```
2.  **Install dependencies**:
    ```bash
    flutter pub get
    ```
3.  **Run the app**:
    ```bash
    flutter run
    ```

## Development

This project follows Clean Architecture principles, separating logic into domain, data, and presentation layers.
